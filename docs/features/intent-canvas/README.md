# Intent Canvas

Status: planned

## Summary

This planned flow would make the first minute of a run focus on placing images and locking intent before the main tools unlock.

## Intended user flow

- click the canvas to import images
- place and resize them directly on the canvas
- choose from guided intent options
- unlock the rest of the app once intent is set

## Main idea

The canvas layout itself becomes an early signal for what the user is trying to make.
