# Native File Browser Dock

Status: planned

## Summary

This planned feature would replace the minimap with a file browser panel so users can drag local images onto the canvas.

## Intended user flow

- choose a folder
- browse images inside the app
- drag an image from the dock to the canvas
- import it into the current run

## Main files

- `desktop/src/index.html`
- `desktop/src/styles.css`
- `desktop/src/canvas_app.js`
