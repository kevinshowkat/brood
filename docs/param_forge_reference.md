# Param Forge Reference Notes

This is an archived reference file. The original Param Forge snapshot is not shipped in this repo.

## What this page keeps

- the historical receipt shape
- the main loop patterns that informed Brood

## Historical receipt shape

Param Forge receipts included:
- request data
- resolved provider/model data
- sanitized provider request and response data
- warnings
- artifact paths
- result metadata such as render time and scores

## Historical loop patterns

- generate -> evaluate -> recommend -> rerun
- plan -> execute -> summarize -> resume
- browse -> compare -> pick a winner

These notes are kept only as background reference.
