# Style Fit Screening Outline

Status: planned

## Summary

This planned feature would let a team compare candidate images against a shared style reference set inside Brood.

## Intended flow

1. Build a style pack from reference images.
2. Add candidate outputs.
3. Review them in a blind grid.
4. Score and shortlist the results.
5. Export a decision packet.

## Core requirements

- local-first storage
- blind review mode
- consistent test set across candidates
- saved review notes and exportable results

## Main areas

- desktop screens for setup and review
- saved sessions and exports in Tauri
- reuse of existing Brood run metadata where possible
