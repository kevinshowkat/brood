# Brood Fit Guide

Use this page to decide whether Brood matches the workflow you need.

| Brood feature | What it helps with |
|---|---|
| Canvas-first editing | Show intent by arranging images instead of rewriting prompts |
| Multi-image tools | Keep target and reference images separate and clear |
| Mother suggestions | Get a suggested next step without starting from scratch |
| Effect tokens | Reuse style or mood information across images |
| Saved receipts and events | Review what happened in a run later |
| macOS desktop runtime | Keep the app and run files local to one machine |

## Brood is a good fit when

- you already have reference images
- you want repeatable runs with saved outputs and receipts
- you prefer a desktop workflow over a browser tool

## Brood is a weaker fit when

- you only want prompt-only image generation
- you need Windows, Linux, or web support today
